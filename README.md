Description
This program is a simple daily task checklist application written in Python.
It allows users to manage their daily tasks using a text-based menu.
Purpose
The program helps users organise their tasks by allowing them to add, view, and remove tasks easily.
Main Features
Add a new task
View all tasks with numbers
Remove a task by selecting its number
Exit the program safely
How to Run the Program
Make sure Python is installed on your computer
Download or clone this repository
Open the file in a Python environment
Run the program and follow the on-screen menu options
